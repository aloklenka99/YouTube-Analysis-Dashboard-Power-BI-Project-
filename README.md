📊 YouTube Data Analysis Dashboard (Power BI Project)

🚀 Project Overview

This project focuses on analyzing a YouTube dataset using Power BI to extract meaningful insights about video performance, audience engagement, and content trends.

The goal is to help content creators and marketers make data-driven decisions to improve engagement and optimize their content strategy.

🎯 Problem Statement

The objective of this project is to:

* Identify patterns in video performance and engagement
  
* Analyze channel growth and subscriber trends

* Understand which content categories perform best

* Provide actionable insights for improving YouTube strategies

📂 Dataset Information

Source: YouTube API dataset (CSV format)

Contains:

* Channel Name & Type
  
* Subscribers

* Views & Uploads

* Channel Rank

* Category-wise data


⚙️ Tools & Technologies Used

📊 Power BI – Data visualization & dashboard creation

🧹 Data Cleaning – Power Query Editor

📈 DAX (Data Analysis Expressions) – Measures & Calculations


🔧 Data Preparation

* Cleaned null/missing values
  
* Transformed data types

* Created calculated columns & measures

* Filtered relevant insights

📊 Key Visualizations Created

✅ Card Visuals:

* Maximum Uploads
  
* Minimum View Rank
* Average Views (Last 30 Days)
  
📈 Line Chart:

* Titles with uploads > 200K
  
🕸️ Radar Chart:

* Channel Type vs Channel Rank
  
🔵 Packed Bubble Chart:

* Category-wise Subscribers
  
📋 Table:

* Channel Type vs Views per Subscriber
  
🌳 Decomposition Tree:

* Rank-wise category analysis


📊 Key Insights

* Entertainment & Music categories have subscriber count

* Some channels have high uploads but low engagement

* Higher “views per subscriber” indicates strong & more audience retention

* Content category plays a major role in channel growth

💡 Business Recommendations

* Focus on high-performing categories like : Entertainment & Music
  
* Improve viewrs retention strategy rather than just increasing uploads
  
* Optimize content using viewer behavior insights
  
* Target makeing quality cointaint for better retention
